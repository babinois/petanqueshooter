Instruction Brick Shooter


Brick Shooter ce jouera avec le clavier,
avec pour touche les fleches du clavier pour diriger (Gauche/Droite) en rouge le vaiseau.
La touche fleche du haut en vert pour tirer sur les blocks qui descende



Sur cet image nous vous montrons:
-En rouge les boutons revenir en arrière.
-En vert pour activer ou désactiver la musique.
-Jaune l'écran du jeux.
-Bleu les boutons pour mettre pause ou reprendre le jeux.
